# Colors

The app uses a dark theme by default. Below are the color values used throughout the UI.

| Element | Light | Dark |
|---|---|---|
| Background | `#fff` | `#0a0a0a` |
| Headings / labels | `#070404` | `#f5f5f5` |
| Weight number | `#060606` | `#f9f9f9` |
| Number boxes | `#d9d9d9` | `#262626` |
| Subtitle / hints | `#6d6969` | `#929292` |
| Nav border | `#000` | `#404040` |
| Chevron icons | `#000` | `#f5f5f5` |
